Responsible Datasets in Context

Star this project on Github

Understanding the social and historical context of data is essential for all responsible data work.

We host datasets that are paired with rich documentation, data essays, and teaching resources, all of which draw on context and humanities perspectives and methods.

We provide models for responsible data curation, documentation, story-telling, and analysis.

Learn more about our Datasets, Team, Mission, and Sponsors. Find us on GitHub , or get in touch.

This website is in development and open to user feedback and discussion. To leave a comment, highlight any text, click “Annotate,” and log into/sign up for Hypothesis.
